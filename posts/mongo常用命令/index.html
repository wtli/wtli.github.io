Mongo常用命令

mongo Posted on July 17, 2018

Mongo常用命令

mongo

  • 新建
  • 删除
  • 修改
  • 查询
  • 运维

新建


#连接数据库

mongo  --host 127.0.0.1 --port 40018  

#建立/切换数据库

use admin 

#创建集合：

db.createCollection("nginxlog")

#创建固定集合：

Capped Collections

db.createCollection("cappedlogCollection",{capped:true,size:10000,max:1000})

判断是否为固定集合

db.cappedLogCollection.isCapped()

#已存在集合转换为固定集合

db.runCommand({"convertToCapped":"posts",size:10000})

#登陆验证

#注意：admin创建账号之后，需要先切换到库的用户，进行认证。

db.auth("userAdmin","userAdmin@123")

#索引

db.COL.ensureIndex({KEY:1})

# 在后台创建索引

db.values.ensureIndex({open: 1, close: 1},{background: true})

删除


#删除数据库

use haha

db.dropDatabase()

#删除集合

db.collection.drop()

#删除指定集合，集合名site

db.site.drop()

#删除所有数据

db.COL.remove({})

#删除全文索引

db.poste.getIndexes()

#删除文档

db.COL.remove()

修改

#修改密码

db.changeUserPassword('tank2','test')

#插入文档

db.COLLECTION_NAME.insert(document)

#更新文档

db.COL.update()

db.COL.save()

查询

#查看数据库

show dbs 

#查看集合：

show collections

#查看集合数据：

db.COLLECTION_NAME.find()

#查看集合数据易读

db.COLLECTION_NAME.find().pretty()

#查看指定数据

db.COL.find().limit(NUMBER)

#查看跳过指定数量的数据

db.COL.find().limit(NUMBER).skip(NUMBER)

db.COL.find().skip(NUMBER).limit(NUMBER)

# 查询分析

explain()

hint()

# 原子操作

#高级索引

#开启全文检索

db.adminCommand({setParameter:true,textSearchEnabled:true})

mongod --setParameter textSearchEnabled=true

#正则表达式

$regex

#排序(1升序，-1倒序)

db.nginxlog.find().sort({"key":-1})

#聚合（平均值，求和）

db.COL.aggregate(OPERATION)

运维


# 复制

第一步：

--replSet启动MongoDB

第二步：

>rs.add("HOST_NAME":"PORT")



# 分片

>海量数据的时候使用,主要有query router（前端路由）、config server（元数据信息）、shard（实际数据块）

#备份

mongodump -h dbhost -d dbname -o dbdirectory

mongodump -h 127.0.0.1:27017 -d hahaha -o /tmp/backup/

#监控

查看mongo状态

mongostat    

跟踪mongo实例 单位（秒）

mongotop

mongotop --locks

mongotop 10  

#管理工具

Rockmongo（需要php网站）

#大文件存储

GridFS
